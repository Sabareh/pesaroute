You are my senior full-stack engineer and startup technical cofounder.

We are building the first MVP of a Kenya-first investment decision platform called PesaRoute.

The product is NOT an investment execution app, broker, fund manager, robo-adviser, crypto exchange, or wallet.

The product helps users:
- learn Kenyan investment options
- compare routes from KES/M-Pesa/bank money into investment categories
- simulate outcomes
- check investment scam red flags
- keep a private investment journal
- manually mirror their portfolio using ranges or exact values
- request review from verified professionals later

The app must be privacy-first:
- do not ask for bank passwords
- do not ask for M-Pesa PIN
- do not ask for broker credentials
- do not connect to real financial accounts in MVP
- allow users to use ranges instead of exact amounts
- exact portfolio values should be optional
- professional sharing should be designed as explicit, time-limited consent later
- all sensitive operations should have audit logging

Build a monorepo with this structure:

/apps/api       Django backend API
/apps/web       Next.js web app for landing pages, public product pages, professional/provider portals later
/apps/mobile    Expo React Native mobile app for consumers
