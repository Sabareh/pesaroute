We need to update PesaRoute’s UI design direction again.

The user wants the app and web app to be inspired by the Apple Developer design page:
https://developer.apple.com/design/

The desired design language is:
- Apple Human Interface Guidelines-inspired
- Liquid Glass / Liquid Surface inspired, but not generic glassmorphism
- premium, clean, modern, professional
- system typography like Apple
- minimal AI visuals
- no robot/neural-network imagery
- no crypto/trading/casino aesthetic
- no overly futuristic dashboard look
- no dark neon glassmorphism

Rename the design system to:

MaliPrime Liquid

Product context:
PesaRoute is a Kenya-first investment decision platform. It helps users learn, simulate, compare investment routes, keep a private investment journal, manually mirror their portfolio, and request review from verified professionals.

PesaRoute does NOT:
- hold user investment money
- execute investments
- act as a broker
- act as a fund manager
- ask for M-Pesa PIN
- ask for bank passwords
- ask for broker credentials
- promise returns
- give unlicensed investment advice

Design language:
- Apple-like polish
- system fonts
- light mode first
- soft grey backgrounds
- white elevated panels
- liquid-style pill controls
- very subtle translucent surfaces only where useful
- soft shadows
- thin borders
- gentle gradients
- crisp typography
- rounded, tactile controls
- professional financial trust

Important:
Do not use heavy glassmorphism everywhere.
Do not make cards transparent over busy backgrounds.
Do not use neon cyber visuals.
Do not add AI robot imagery.
Do not add neural network graphics.
Do not make the app look like crypto trading, sports betting, or a stock casino.

Typography rules:
Use Apple-style system font stacks.

Web CSS font stack:
font-family:
  -apple-system,
  BlinkMacSystemFont,
  "SF Pro Display",
  "SF Pro Text",
  "Inter",
  "Geist",
  "Segoe UI",
  Roboto,
  Helvetica,
  Arial,
  sans-serif;

Important font restriction:
Do not download, bundle, or redistribute Apple SF Pro or New York font files.
Use system fonts only.
On iOS, use native system font.
On Android, use Roboto or Inter fallback.
For web, use -apple-system/BlinkMacSystemFont first.
