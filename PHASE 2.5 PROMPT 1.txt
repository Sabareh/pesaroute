PHASE 2.5 PROMPT 1 — Replace MaliGlass with MaliPrime design system

We are doing a UI/UX overhaul for PesaRoute.

The previous direction used glassmorphism / MaliGlass. We are replacing it completely with a premium modern fintech design language called MaliPrime.

Product context:
PesaRoute is a Kenya-first investment decision platform. It helps users learn, simulate, compare investment routes, keep a private investment journal, mirror their portfolio manually, and request review from verified professionals.

PesaRoute does NOT:

* hold money
* execute investments
* ask for M-Pesa PIN
* ask for bank passwords
* ask for broker credentials
* promise returns
* give unlicensed investment advice

New design direction:

* premium modern fintech
* Apple-style system typography
* light mode first
* solid cards, not glass
* subtle shadows
* subtle gradients only as accents
* clean professional layout
* minimal illustrations
* no heavy blur
* no glassmorphism
* no neon trading aesthetic
* no crypto/casino look
* no AI/robot/neural-network visuals
* no generic “AI startup” imagery
* no overly futuristic dashboards

Design inspiration:

* Apple Wallet simplicity
* Stripe polish
* Linear clarity
* Wise/Revolut trust
* modern wealth dashboard
* premium banking app

Do not copy any specific brand, logo, or proprietary style.

Typography direction:
Use modern system sans-serif fonts.

Web font stack:
font-family:
-apple-system,
BlinkMacSystemFont,
"SF Pro Display",
"SF Pro Text",
"Inter",
"Geist",
"Segoe UI",
Roboto,
Helvetica,
Arial,
sans-serif;

Important:

* Do not download, bundle, or redistribute Apple SF font files.
* Use system font stacks only.
* On iOS, use the native system font.
* On Android, use Roboto by default or Inter if already configured safely.
* If adding custom fonts, use open-source fonts like Inter or Geist only.
* Do not add heavy font packages unnecessarily.

Visual identity:

* default background: warm off-white / soft grey
* surfaces: clean white
* surface alt: very light blue-grey
* text: deep navy / charcoal
* secondary text: muted slate
* primary accent: cobalt blue
* wealth accent: emerald green
* warning: amber
* danger: red
* border: light grey
* gradients only for hero cards, primary CTAs, and small premium highlights

background: white

Design principles:

1. Calm over flashy.
2. Trust over hype.
3. Professional over futuristic.
4. Education over trading.
5. Privacy over data hunger.
6. Kenyan context, global polish.
7. Mobile-first readability.
8. No fake precision in investment data.
9. No design that makes investing feel like gambling.
